Database Information

Setup Required:

  • ✓ MySQL Server running on localhost
  • ✓ Database: student_db
  • ✓ Table: students (id, name, email)
  • ✓ PHP with MySQLi extension enabled

Quick Start:

  2. Click "Create Database & Table" button
  3. Add new student records
